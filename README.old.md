# COM00151M final project repository.
# It contains source code for an HTTP Microservices REST API Client application.
 
## Running Locally

Make sure you have [Node.js](http://nodejs.org/) installed.

```sh
git clone https://github.com/lnb512/https-github.com-lnb512-com00151m-http-api-microservice-client.git
cd repository folder
npm install 
npm start
```

# Your client app should now be running locally.
Load http://localhost:8080 on your client (browser) to send request to HTTP Microservice.
The HTTP Problem Details Response will be sent to your terminal console. 

# Use ca also use Postman API client to send client Requests to the HTTP Microservice REST API
Make sure you have [Postman](https://www.postman.com/downloads/) installed. 
Send a GET request to http://localhost:3000 with relevant parameters and the response will appear below your request.
